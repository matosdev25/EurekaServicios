# Eureka Servicios - Landing Page PRD

## Original Problem Statement
Crear una landing page responsive (desktop + mobile) en español para "Eureka Servicios", empresa de reparaciones del hogar y negocio en Ciudad de Panamá. Estética limpia, bordes redondeados, sombras suaves, paleta verde/crema.

## User Personas
- **Propietarios de viviendas**: Necesitan reparaciones urgentes del hogar (filtraciones, goteras, humedad)
- **Dueños de negocios**: Requieren mantenimiento profesional para sus establecimientos
- **Administradores de edificios**: Buscan servicios confiables para múltiples propiedades

## Core Requirements (Static)
1. Landing page single-page responsive
2. Header sticky con navegación y menú hamburguesa mobile
3. Hero section con CTA a WhatsApp
4. Secciones: Socio confiable, Problemas, Servicios, Bricolaje, Reseñas, CTA
5. Footer con enlaces y contacto
6. Botones WhatsApp con texto prellenado
7. Paleta verde (#1A4D2E) / crema (#FDFCF8)
